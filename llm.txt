# Claude Parallel Implementation Workflow

> A shell-based tool that runs 3 parallel Claude Code implementations for feature requests and automatically selects the best one.

This tool streamlines feature development by creating parallel git worktrees, running Claude Code instances simultaneously, reviewing all implementations, and creating a draft PR with the winning solution. It leverages git worktrees to enable concurrent development without conflicts.

## Main Script

- [parallel-impl.sh](parallel-impl.sh): Main shell script entry point that orchestrates the entire parallel implementation workflow including worktree creation, parallel Claude execution, review, and PR creation.
